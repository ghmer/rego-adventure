## Project Information

This is a standalone web-based editor for creating and editing quest JSON files used in the Rego Adventure project. The Quest Editor provides an interface for managing quest metadata, storylines, hints, tests, and other quest-related content.

## Usage

The Quest Editor is a client-side web application consisting of:
- [`index.html`](index.html) - Main HTML structure
- [`app.js`](app.js) - JavaScript application logic
- [`styles.css`](styles.css) - CSS styling

To use the editor, simply open `index.html` in a modern web browser. No server or build process is required.
